# ChronoForge - 13-Phase Productivity Calendar (MVP)

## Overview
ChronoForge is a **cyclical productivity system** based on the 13-phase framework. Users can:
- Track each phase/day of their cycle
- Record metrics: Mood, Task Completion, Focus, Notes
- See real-time progress and stats
- Receive motivational guidance per phase/day

## Features
- Dynamic calendar generation (JS-driven)
- Dark theme header & interface
- Emoji mood picker
- Task completion selector (0-10 scale)
- Focus slider with visual scale
- Stats summary: Avg Energy, Total Output, Avg Quality, Consistency %
- Phase & cycle progress bars
- Real-time metrics updates

## File Structure
- index.html
- assets/css/style.css
- assets/js/engine.js
- assets/fonts/



## Usage
1. Open `index.html` in a browser.
2. Click any day in the calendar to view/edit metrics.
3. Select mood, task completion, focus, and optional notes.
4. Observe real-time progress bars and stats update.
5. Metrics are stored per-day for your session.
